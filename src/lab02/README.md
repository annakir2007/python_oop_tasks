# Реализованная коллекция

В рамках лабораторной необходимо реализовать **контейнер для хранения объектов**, созданных в **ЛР-1**.

Контейнер управляет группой объектов выбранной предметной области.

Пример:

```
Student -> StudentGroup
Book -> Library
Product -> ProductCatalog
Car -> Fleet
Player -> PlayerList
```

### Реализовано

Контейнерный класс должен:

* хранить **множество объектов**
* управлять добавлением и удалением
* предоставлять доступ к объектам
* позволять итерироваться по коллекции

# Задание на 3

## Требования

1. Использовать **класс из ЛР-1**.
2. Реализовать **класс коллекции**, который хранит объекты этого класса.

Например:

```
Student -> StudentCollection
Book -> Library
Product -> ProductCatalog
```

3. Внутри контейнера хранить список объектов.

```
self._items = []
```

4. Реализовать методы:

* `add(item)` — добавить объект
* `remove(item)` — удалить объект
* `get_all()` — вернуть список объектов

5. Проверять тип добавляемых объектов.

Например:

```
можно добавлять только Student
```

---

## Демонстрация

В `demo.py` должно быть:

* создание нескольких объектов
* добавление их в коллекцию
* вывод всех элементов
* удаление элемента
* повторный вывод коллекции

---

## Проверки

* нельзя добавить объект неправильного типа
* удаление должно корректно работать
* коллекция должна хранить только допустимые объекты

# Задание на 4

## Требования

0. Реализовать всё из **задания на 3**

1. Реализовать методы:

* `find_by_*()` — поиск по одному из атрибутов

Например:

```
find_by_name()
find_by_id()
find_by_title()
```

2. Реализовать:

```
__len__()
```

чтобы можно было писать:

```
len(collection)
```

3. Реализовать:

```
__iter__()
```

чтобы можно было писать:

```
for item in collection:
```

Пример:

* нельзя добавить дубликат
* нельзя добавить объект с тем же id
* нельзя добавить объект с одинаковым именем

## Демонстрация

В `demo.py` показать:

* поиск элемента

# Задание на 5

## Требования

0. Реализовать всё из заданий **на 3 и на 4**

1. Реализовать **индексацию коллекции**

```
collection[0]
collection[2]
```

через:

```
__getitem__()
```

2. Реализовать **удаление по индексу**

```
remove_at(index)
```

3. Реализовать **сортировку объектов**.

Например:

```
sort_by_name()
sort_by_price()
sort_by_date()
```

или универсально:

```
sort(key)
```

4. Добавить **логические операции над коллекцией**

Например:

```
get_active()
get_available()
get_expensive()
```

Метод должен возвращать **новую коллекцию**.
